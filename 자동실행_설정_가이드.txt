============================================================
  Windows 자동 실행 설정 가이드
============================================================

[방법 1] 시작프로그램 폴더에 바로가기 추가 (권장)
------------------------------------------------------------
1. Windows + R 키를 누릅니다
2. "shell:startup" 입력 후 엔터
3. 열린 폴더에 "자동실행.bat" 파일의 바로가기를 복사합니다
4. PC 재부팅 시 자동으로 HMI가 실행됩니다

장점: 설정이 간단하고 안전함
단점: 사용자 로그인 후 실행됨


[방법 2] 작업 스케줄러 사용 (고급)
------------------------------------------------------------
1. 작업 스케줄러 실행
   - Windows 검색에서 "작업 스케줄러" 입력

2. 작업 만들기
   - 우측 메뉴에서 "작업 만들기" 클릭

3. 일반 탭 설정
   - 이름: ESS HMI 자동실행
   - "가장 높은 수준의 권한으로 실행" 체크

4. 트리거 탭
   - "새로 만들기" 클릭
   - "작업 시작": 시작할 때
   - "확인" 클릭

5. 동작 탭
   - "새로 만들기" 클릭
   - 프로그램/스크립트: [HMI 폴더]\자동실행.bat
   - 시작 위치: [HMI 폴더 경로]
   - "확인" 클릭

6. 조건 탭
   - 전원 관련 옵션 해제

7. "확인" 클릭하여 작업 저장

장점: 부팅 즉시 실행, 관리자 권한으로 실행 가능
단점: 설정이 복잡함


[방법 3] 레지스트리 등록 (전문가용)
------------------------------------------------------------
※ 주의: 레지스트리 수정은 신중하게 진행하세요

1. Windows + R 키를 누릅니다
2. "regedit" 입력 후 엔터
3. 다음 경로로 이동:
   HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run
4. 우클릭 > 새로 만들기 > 문자열 값
5. 이름: ESS_HMI
6. 값: [HMI 폴더 경로]\자동실행.bat
7. PC 재부팅


[자동실행 해제 방법]
------------------------------------------------------------
방법1: shell:startup 폴더에서 바로가기 삭제
방법2: 작업 스케줄러에서 해당 작업 삭제
방법3: 레지스트리에서 해당 항목 삭제


[문제 해결]
------------------------------------------------------------
Q. 자동실행이 안 됩니다
A. autostart.log 파일을 확인하여 오류 내용을 확인하세요

Q. 부팅 시 창이 여러 개 뜹니다
A. 정상입니다. 시뮬레이터, 백엔드 서버 2개의 창이 실행됩니다

Q. 자동실행을 일시적으로 중지하고 싶습니다
A. Shift 키를 누른 상태로 부팅하면 시작프로그램이 실행되지 않습니다
